# easy-form-validator

**How to Install**

`npm i easy-form-validator`

**Validate email**

`import { isEmailValid } from 'easy-form-validator'`

Usage : _isEmailValid('arulgetsolute@gmail.com')_

result: _**True | false**_


**Validate name**

`import { isEmailValid } from 'easy-form-validator'`

Usage : _isEmailValid('arulmozhimanikandan')_

result: _**True | false**_
